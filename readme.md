MSDN Key Manager
======

Export xml keys as xml, leave as 'KeysExport.xml'

Place in working directory if building through VS, or in /src/keymng if running via dnx

Build/Run crossplatform to DNX core

	//dvmn use default -r coreclr
	dnu restore
	dnx . run
